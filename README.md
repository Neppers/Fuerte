# Fuerte

## Requirements

Or at least what it has been built with

- Node 0.10.26+
- MongoDB 2.6.3+

## Installation (OSX)

Assuming both Node and MongoDB are already installed

```mkdir data && mongod --dbpath data```

```npm install```

```bower install```

```gulp styles && gulp scripts && gulp fonts```

```npm start```

Default port is 3000
